🕯️ Fase 1: Fundamentos HTML

Nesta fase, você aprenderá sobre as estruturas básicas do HTML, suas tags e a importância da semântica.

O HTML (HyperText Markup Language) é a linguagem usada para criar a estrutura de uma página na internet. Ele funciona como o esqueleto de um site, organizando textos, imagens, links e outros elementos que aparecem na tela.

Usar corretamente as tags semânticas torna o site mais acessível e compreensível — tanto para pessoas quanto para buscadores mágicos da web. Aprender HTML é o primeiro passo para se tornar um verdadeiro autor ou autora da magia digital.
